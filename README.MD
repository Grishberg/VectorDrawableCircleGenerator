Скрипт для генерации pathData для Android Vector Drawable файлов,
создайте/измените `in.xml` с нужным списком фигур и запустите `svg_path_generator.py`

Структура `in.json`:
    Корневой элемент:
    `shapes` - представляет собой масив описателей фигур

    Описание фигуры:
    `name` - название строковой константы для которой создается pathData
    `type` - тип фигуры (`rect`, `circle`)

    Для типа фигуры `rect`:
    `x` - X левого верхнего угла в dp
    `y` - Y левого верхнего угла
    `w` - ширина прямоугольника
    `h` - высота прямоугольника
    `r` - радиус скругления вершины

    Для типа фигуры `circle`:
    `x` - X центра в dp
    `y` - Y центра
    `r` - радиус окружности
